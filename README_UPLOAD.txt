LUXES EXECUTIVE - LOCAL CARS.JS VERSION

Эта версия работает даже двойным кликом по index.html.

Важно:
- Сайт НЕ читает cars.csv напрямую.
- Сайт читает cars.js.
- cars.csv и cars.xlsx остаются как база/таблица.
- Для обновления сайта нужно обновить cars.js.

Внутри есть файл convert_csv_to_cars_js.html:
1. Открой cars.csv.
2. Скопируй всё содержимое.
3. Открой convert_csv_to_cars_js.html.
4. Вставь CSV.
5. Нажми "Сделать cars.js".
6. Скопируй результат и замени файл cars.js.

Фотографии должны лежать в assets/cars/.
В колонке Фото должно быть только имя файла, например:
lexus-gx460-2016.jpg
